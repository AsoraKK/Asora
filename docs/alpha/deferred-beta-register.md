# Deferred Beta Register

The following work remains in the repository but is explicitly excluded from the web/API Alpha launch gate:

- Android and iOS production distribution.
- Play Console and TestFlight readiness.
- Firebase mobile distribution.
- Android signing and production mobile certificate-pin evidence.
- Store listing, screenshots, review notes, and public rollout configuration.
- Mobile crash-symbol upload and device-farm release certification beyond existing static/unit coverage.
- Production billing/payment-provider activation. Alpha Premium and Black are audited manual grants labelled `Alpha access`.
- Automatic cohort progression or public self-service invitations.
- Autonomous operational-agent mutation, deployment, rollback, moderation, credential, threshold, cohort, schema, or infrastructure actions.

Deferral does not authorize removal of mobile code or internal Asora infrastructure identifiers.
